## About the project

This project is a web application built using Next.js(page based routing), TypeScript, Tailwind CSS, and Supabase. It allows users to view and manage notices, with features such as filtering by category and priority.

## to run the project locally, follow these steps:

1. Clone the repository to your local machine using `git clone <repository-url>`.
2. Navigate to the project directory: `cd reno-cred`
3. Install the dependencies: `npm install`
